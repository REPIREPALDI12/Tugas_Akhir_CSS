Materi Frontand



HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman website Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange.

selengkapnya

CSS

CSS (Cascading Style Sheet) adalah bahasa yang digunakan untuk mengatur tampilan pada HTML. Dengan CSS, kita dapat memberikan warna background, border, mengatur posisi elemen, ukuran font dan lain sebagainya.

selengkapnya

Java SKRIPT

Javascript adalah bahasa pemrograman (berjalan pada web browser) yang digunakan untuk membuat halaman web yang lebih interaktif. Dengan javascript, kita bisa membuat banyak fitur pada halaman web.

selengkapnya

bootstrap

Bootstrap adalah sebuah framwork yang di jadikan senjata andalan bagi para frontand developer saat menciptakan website. Tak lain karena Bootstrap bisa membantu mereka membuat website dengan cepat dan mudah.

selengkapnya

Materi Backand



PHP

PHP adalah bahasa open-source yang banyak digunakan oleh pemrograman atau pengembangan website (web development). Bahasa ini umumnya dijalankan dalam komunikasi sisi server, dan saat ini didukung oleh hampir semua sistem.

selengkapnya

Golang

Golang merupakan bahasa pemrograman yang didirikan oleh Google pada tahun 2009 oleh Robert Griesemer, Rob Pike dan Ken Thompson. Bahasa ini biasanya digunakan untuk keperluan server side.

selengkapnya

React JS

React JS adalah library JavaScript yang biasanya digunakan untuk membangun sebuah UI dalam suatu website atau aplikasi web. Jadi, React JS bisa dianggap seperti perpustakaan yang berisi berbagai kode JavaScript yang sudah tertulis (pre-written).

selengkapnya

Phyton

Python adalah bahasa pemrograman serbaguna yang bisa dijalankan pada hampir semua arsitektur sistem, dan bisa digunakan untuk berbagai aplikasi di banyak bidang, mulai dari web development hingga machine learning.

selengkapnya
